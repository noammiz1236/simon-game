Simon Game – JavaScript Implementation
Fully developed by Noam Mizrahi
Description:

This project is a full implementation of the classic “Simon” memory game using JavaScript and jQuery.
The system generates an increasing sequence of colors, and the player must repeat the sequence correctly.
A wrong input resets the game; correct input advances the level.

Features

Random color generation each round

Color sequence animation

Player input handling

Real-time comparison between player sequence and system sequence

Full game reset on mistake

Game starts by pressing any key

How It Works

The game starts when the user presses a key

A random color is added to the sequence

The user clicks buttons to repeat the sequence

After every click, the game checks for correctness

Mistake → Reset

Correct sequence → Level increases and a new color is added
